How does it work?

There are two ways to hook up a target

As a <script> tag

<script type="text/javascript" src="path_to_weinre_script#unique_id"/>

As a bookmarklet

A bookmark with javascript that adds
the javascript tag
(this sounds a lot like the yo dawg-meme)

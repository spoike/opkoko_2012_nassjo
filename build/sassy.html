Lets check out some code 

#cssrules {
    font-size: 20pt;
}
div.content {
    font-size: 10pt;
    background-color: #bada55;
    margin: 20px;
}
div.content pre {
    margin: 10px;
    text-shadow: 2px 2px 2px #333;
}

That's a lot of code…

How about we add some variables?

$bgcolor: #bada55;
$fsize: 20pt;
 
#cssrules {
    font-size: $fsize;
}
div.content {
    font-size: 10pt;
    background-color: $bgcolor;
    margin: 20px;
}
div.content pre {
    margin: 10px;
    text-shadow: 2px 2px 2px #333;
}

Let's put in some arithmetic

$bgcolor: #bada55;
$fsize: 20pt;
$common_margin: 10px;
 
#cssrules {
    font-size: $fsize;
}
div.content {
    font-size: $fsize / 2;
    background-color: $bgcolor;
    margin: $common_margin * 2;
}
div.content pre {
    margin: $common_margin;
    text-shadow: 2px 2px 2px #333;
}

How about some lovely nesting?
